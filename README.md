# Ciel

## サイト概要
ビデオ通話ができるマッチングアプリ

### サイトテーマ
No need to meet up

### テーマを選んだ理由
昨今、コロナウイルスの影響で過度な外出を控える若者が増えていると考える。
外出をしなくても出会うことができ、ビデオ通話を通して交流できるように。

### ターゲットユーザ
10代後半〜30代後半

### 主な利用シーン
家にいながら異性と交流したい時
