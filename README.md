# Ciel

## サイト概要
訪日外国人と日本を案内できる人のマッチングアプリ

### サイトテーマ
Local Japan

### テーマを選んだ理由
海外旅行に行った際、現地に住む友人に案内してもらい有意義な旅行になったから

### ターゲットユーザ
訪日外国人

### 主な利用シーン
訪日外国人が、日本のディープな居酒屋など、自分達だけでは行けないお店や観光地に行きたい時
